# JavaScript Array Operations Project

This project demonstrates the use of important JavaScript array methods such as `map()`, `forEach()`, `filter()`, `push()`, and `concat()` using a simple employee dataset.

## 🚀 Features

* Display employees with profession "developer" using:

  * `map()`
  * `forEach()`
* Add a new employee object to the array
* Remove employees with profession "admin"
* Concatenate two arrays of employee objects

## 🛠️ Technologies Used

* HTML
* JavaScript

## 📌 How to Run

1. Clone the repository
2. Open `index.html` in any browser
3. Open Developer Console (F12)
4. Click buttons to see outputs
